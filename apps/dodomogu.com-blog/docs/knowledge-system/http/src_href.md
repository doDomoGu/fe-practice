# src 与 href 的区别

| src                                             | href                                      |
| ----------------------------------------------- | ----------------------------------------- |
| src 是 source 的缩写                            | href 是 Hypertext Reference 的缩写        |
| src 用于替换当前元素                            | href 用于在当前文档和引用资源之间确立联系 |
| 在 `<img>`、`<script>`、`<iframe>`中， 表示引入 | 在 `<link>`、`<a>`中，表示链接            |
| 会暂停其他资源的下载和处理                      | 并行下载资源                              |
